# Bin2Dec

Bin2Dec is a Java console application that converts a binary number (base 2)
into its decimal (base 10) equivalent.

## How to Run

1. Compile the program:
   ```
   javac Bin2Dec.java
````

2. Run the program:

   ```
   java Bin2Dec
   ```

3. Enter a binary number using only 0s and 1s

4. Type `quit` to exit

## Example

Input:

```
1011
```

Output:

```
Decimal value: 11
```
